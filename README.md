# My Author Edits

Joomla system plugin that filters category menu items by the logged-in author and optionally includes drafts.

## Contents

- `plugins/system/inartsmyauthorfilter/`: installable plugin source
- `dist/plg_system_myauthoredits-1.1.0.zip`: Joomla install package

## Notes

- The live plugin keeps its internal folder name for stability.
- Visible plugin name is generic and translated through language files.
- Native Joomla updates still need a public update server XML URL.
